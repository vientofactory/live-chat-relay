# How to use

## clone this project

```
git clone https://github.com/vientofactory/live-chat-relay.git
```

## install dependencies

npm:

```
npm install
```

yarn:

```
yarn
```

## build

```
npm run build
```

## copy html template

```
cp -r src/server/static dist/server
```

## start server

```
npm run start
```
